mutex *  mut,

inline

Constructs a Velocity object. Distance is a pointer that should increase in value as the caller moves. When Velocity detects that distance has changed from its initial value, it will begin monitoring speed.

Parameters
distanceA pointer to the distance covered by the client class, which Velocity expects to change over time
mutA mutex shared with Velocity that should be locked when this velocity or distance is about to be deleted
