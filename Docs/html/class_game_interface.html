Detailed Description

The pure virtual base class that will serve as an interface to every object in the game world. It defines all methods by which any class can talk to another. Since KeyInputRegister expects an instance of GameInterface for member function calls, of Any class that wishes to make its its methods available for callback from input events should add a pure virtual version of said functions to GameInterface, inherit from GameInterface, and implement.
