Detailed Description

template<typename N>
struct Position< N >

Position<> is a simple vector type with the useful feature of storing all its previous states, back through time. Thus an object using Position to store location information will be able to retrace its steps. Note: Classes with a Position data member will typically want to have a pointer, instead of holding the Position locally. This is because many objects in the World may not actually have a physcical Position in space, in which case they can just hold a null pointer.
